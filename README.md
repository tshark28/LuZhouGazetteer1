# 直隶泸州志 - 清道光年间（1821-1850年）
- Local historical gazetteer of Luzhou, compiled or published during the Daoguang reign (1821–1850) of the Qing Dynast

## 书籍简介

《直隶泸州志》是四川省泸州市（古称直隶泸州）的清代地方志，由清代道光年间编纂完成。本志书全面记载了泸州的历史沿革、地理环境、政治制度、经济状况、文化风貌等内容，是研究泸州乃至川南地区历史文化的珍贵史料。

## 编纂信息

- **编纂年代**: 清道光年间（1821-1850年）
- **全书卷数**: 共十一卷
- **内容涵盖**: 舆地、建置、官师、选举、人物、食货、武备等

## 历史价值

1. **地理研究**: 详细记录泸州山川河流、行政区划，为清代川南地理研究提供重要资料
2. **政治制度**: 记载清代官制、科举制度，反映当时政治运作机制
3. **经济状况**: 记录赋役、人口、物产等经济数据，是研究清代地方经济的第一手资料
4. **文化风貌**: 记载祠庙、古迹、人物等，体现泸州历史文化底蕴
5. **人物传记**: 收录大量乡贤、列女、孝友等人物事迹，保存珍贵的人物史料
